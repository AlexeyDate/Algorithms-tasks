В данной задаче ваша цель - реализовать симуляцию параллельной обработки списка задач.
Такие обработчики (диспетчеры) есть во всех операционных системах.

У вас имеется n процессоров и последовательность из m задач. 
Для каждой задачи дано время, необходимое на её обработку. 
Очередная работа поступает к первому доступному процессору (то есть если доступных процессоров несколько, то доступный процессор с минимальным номером получает эту работу).
